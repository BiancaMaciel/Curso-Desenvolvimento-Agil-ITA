# Curso-Desenvolvimento-Agil-ITA

Este curso consiste em oito módulo sendo eles:

1. Orientação a Objetos com Java - duração de 6 semanas. 
2. TDD – Desenvolvimento de software guiado por testes - duração de 4
semanas;
3. Desenvolvimento Ágil com Java Avançado - duração de 4 semanas;
4. Princípios de desenvolvimento ágil de software - duração de 4 semanas;
5. Desenvolvimento ágil com padrões de projeto - duração de 4 semanas;
6. Técnicas Avançadas para projeto de software - duração de 4 semanas;
7. Arquitetura de Software em projetos ágeis - duração de 4 semanas;
8. Trabalho de Conclusão de Curso: Aplicativo para web com componente
gamificado - duração de 8 semanas.
